# quickpascal_l25
Petit jeu L25 écrit en QuickPascal 1.0 de Microsoft sous DOS.
Le but du jeu est de remplir 25 cases (5 x 5) en se déplaçant comme un cheval au échec sans jamais touchés deux fois la même case.
